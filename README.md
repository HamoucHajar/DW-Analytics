# DW-Analytics

## Project Description
This project demonstrates the design and implementation of a **Data Warehouse for agricultural data** using a **Star Schema**.  
It integrates **Pentaho Data Integration (PDI)** for ETL processes and **Power BI** for data visualization and analytics, enabling **data-driven decision-making in agriculture**.

## Technologies Used
- **Pentaho Data Integration (PDI)** ‚Äì for extracting, transforming, and loading agricultural data  
- **Data Warehouse Star Schema** ‚Äì fact and dimension tables for optimized querying  
- **Power BI** ‚Äì for interactive dashboards and reporting  
- **Source Data** ‚Äì agricultural datasets from CSV files or databases
